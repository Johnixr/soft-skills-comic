# 软技能生存指南 — 机器猫漫画版

> 《软技能生存指南》机器猫漫画版 — 基于 John Sonmez 原著 [*Soft Skills: The Software Developer's Life Manual*](https://www.amazon.com/Soft-Skills-Software-Developers-Manual/dp/1617292397) 的 AI 时代中国语境改编，一人公司（OPC）视角的 71 章机器猫漫画连载。

---

## 📖 内容说明

本项目仅为**个人学习、研究与技术实验用途**，由 AI 辅助将原著内容以机器猫漫画风格进行重新演绎，并适配中国语境与本土平台场景。

- 原著作者：**John Sonmez**（《Soft Skills: The Software Developer's Life Manual》）
- 改编方式：AI 辅助生成，基于原著框架与思想进行中国语境改编
- 风格：机器猫（哆啦A梦）漫画风格，场景化叙事
